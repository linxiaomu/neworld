1. 格式化快捷键：**Ctrl+Alt+L**
2. 快速删除不用的import ：**Ctrl+Alt+o**
3. 向下插入新行：**Shift+Enter**
4. 上插一行：**Ctrl+Enter**
5. 可以显示最近编辑的文件列表：**Ctrl＋E**
6. 变换光标：**shift+insert(0)**
7. getter和setter方法：**Alt-Insert**




eclipse 格式化代码快捷键：Ctrl+Shift+F（失效是搜狗输入法占用）
